# itemschema.github.io

Landing page for [itemschema.com] - In-game item APIs

### Tech

*  Node - ">= 16.0.0"
*  Parcel - build tool
*  [lucide] - icon library

### Installation

To run locally
```bash
npm run dev
```

### Deployment

Build for prod
```bash
npm run predeploy
```

Push to prod
```bash
npm run deploy
```
